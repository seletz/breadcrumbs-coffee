breadcrumbs-coffee
==================


Build the project
-----------------


```bash
    $ cake watch
```


Run the tests
-------------

```bash
    $ python -c "from SimpleHTTPServer import test; test()"
    Serving HTTP on 0.0.0.0 port 8000 ...
```

Browse to http://localhost:8000/test-js/suite.html
